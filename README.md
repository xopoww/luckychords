# luckychords
Small util to generate random chords

For each chord picks random root from C major scale and one of the following qualities:
- major
- minor
- diminished
- augmented

Usage:
```
$ python luckychords.py 
Your lucky chords are:
1. A dim  (A, C, D#)
2. E      (E, G#, B)
3. B aug  (B, D#, G)
4. Gm     (G, A#, D)
```
